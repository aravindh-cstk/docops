# [Search Content] - Run a Quick Search

This page explains how to use Contentstack’s search bar (including the **K** keyboard shortcut) to quickly find entries containing specific keywords. It’s intended for Contentstack users who need to locate content entries fast, and should be used when you want to run a simple keyword search before applying additional filters.

## Run a Quick Search

Consider a scenario where you are looking to quickly bring up all the entries that contain the name “**John Smith**” within their fields.

Perform the following steps:
- Navigate to the search bar (or press “**K**”)  located on the header tab of the Contentstack site. You can also use the shortcut key “K” for both Windows OS and Mac OS users to access search.
- Select the **Entries** option from the left-hand side drop-down menu, based on your requirement.![Image-Entries-Search.jpg](https://images.contentstack.io/v3/assets/blt23180bf2502c7444/blt58e16657295acd86/639b2fda2e75dc108bf88d82/Image-Entries-Search.jpg)
- Type in the keywords “John Smith” and press **Enter**.

The system will display all the entries of your stack that contain the term “**John Smith**”.

You can further refine the search results using Filters.

## Common questions

**How do I open the search bar quickly?**  
Press “**K**” or navigate to the search bar located on the header tab of the Contentstack site.

**Can I search for something other than entries?**  
Yes. Select the appropriate option from the left-hand side drop-down menu, based on your requirement.

**What happens after I press Enter?**  
The system will display all the entries of your stack that contain the term you searched for.

**How can I narrow down the results?**  
You can further refine the search results using Filters.

# [Author Content] - JSON Rich Text Editor for Content Managers

This page explains how the **(JSON RTE)** in Contentstack supports content managers in creating and formatting content without HTML, including built-in formatting tools and drag-and-drop content blocks. Read this when you are authoring content in Contentstack and want to understand the editor’s capabilities and workflow.

## JSON Rich Text Editor for Content Managers

The [JSON Rich Text Editor](../../developers/json-rich-text-editor/about-json-rich-text-editor.md)
  ** (JSON RTE)** in Contentstack simplifies content creation by eliminating the need for HTML. It offers intuitive styling options and flexible content arrangement, making it easier for content managers to focus on creating compelling content.

## Format Without HTML

  JSON RTE does not include an HTML code view. This removes the need to manually handle HTML source code, allowing content managers to style text using built-in formatting tools. These tools are available in the menu bar at the top of the editor field.

## Drag and Drop Content Blocks

  Each content element—such as paragraphs, images, or lists—is treated as an individual block. Content managers can easily rearrange these blocks by dragging and dropping them within the editor, without copying or pasting.

## Common questions

### Does JSON RTE include an HTML code view?
No. JSON RTE does not include an HTML code view.

### Where are the formatting tools located?
These tools are available in the menu bar at the top of the editor field.

### Can content blocks be rearranged in the editor?
Yes. Content managers can easily rearrange these blocks by dragging and dropping them within the editor, without copying or pasting.

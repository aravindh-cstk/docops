# [Author Content] - Rename Asset Versions

This page explains how to rename an existing asset version in Contentstack. It is intended for content managers and authors who manage assets and need clearer version labels for easier identification and organization.

## Rename Asset Versions

You can rename an existing version of an [asset](./about-assets.md) in Contentstack to enhance and streamline content management. Renaming versions allows you to assign meaningful labels, making it easier to identify and manage specific versions of your assets effectively.

To rename an existing asset version, log in to your [Contentstack account](https://www.contentstack.com/login), and perform the following steps:
- Go to your [stack](../../developers/set-up-stack/about-stack.md) and click the “Assets” icon in the left navigation panel.
- Navigate to the asset you want to modify.
- Click the **version** **dropdown **in the top-right corner of the page.![Version_Dropdown.png](https://images.contentstack.io/v3/assets/blt2d43f51baca745a8/blt9a60280c9aa19329/674d699b4b0ed1249de1dc8a/Version_Dropdown.png)
- From the dropdown menu, view the list of all asset versions. Click the **Rename** icon next to the version you want to name.![image2.png](https://images.contentstack.io/v3/assets/blt2d43f51baca745a8/bltf0260f12a5d8faad/674d6a8b31c3824dc38d31d5/image2.png)
- Enter the new name for the version. Click the **checkmark **icon or press **Enter **to save your changes.![Rename_Checkbox.png](https://images.contentstack.io/v3/assets/blt2d43f51baca745a8/blt6c8cc4471d53cd95/674d69a12452f957060a742b/Rename_Checkbox.png)

**Note**: Custom version names have a maximum limit of **32 characters**.

## Common questions

### Who can rename asset versions?
Users who can access the stack and the asset in Contentstack and have the necessary permissions to modify assets can rename asset versions.

### Does renaming an asset version change the asset file?
No. Renaming an asset version changes only the version label, not the underlying asset file.

### What is the maximum length for a custom version name?
Custom version names have a maximum limit of **32 characters**.

### Can I rename multiple asset versions?
Yes. You can rename versions individually by selecting each version from the version dropdown and using the **Rename** icon.

# Set canvas without offset


**Method:** `GET`  
**Endpoint:** `/assets/{stack_api_key}/{asset_uid}/{file_uid}/filename.jpg?environment={environment_name}&canvas={width_value},{height_value},x{value},y{value}`

**Parameters:**

| Key | Value | Description |
|-----|-------|-------------|

| canvas | 650,650,x20,y20 | You can pass these values:  - width: Enter the width of the canvas in pixels or percentage. - height: Enter the height of the canvas in pixels or percentage. -  |

| environment | production | Enter the environment scoped to your delivery token. |

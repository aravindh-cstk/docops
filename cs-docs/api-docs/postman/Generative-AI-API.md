# Generative AI API


The Contentstack CMA Postman Collection is a set of preconfigured REST API requests that will make it easy for you to get started with the [Generative AI APIs](../api-detail/generative-ai-api.md) and try out our API requests through the [Postman](https://www.getpostman.com/) REST client.

To use the API Request, you will need to authenticate yourself either with a [Management Token (highly recommended)](../../cs-docs/developers/create-tokens/about-management-tokens.md) or an [Authtoken.](../api-detail/content-management-api.md#how-to-get-authtoken)   
Read more about it in [Authentication.](../api-detail/content-management-api.md#authentication)

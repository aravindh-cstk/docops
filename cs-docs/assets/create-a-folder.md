# [AM2.0] - Create a Folder

This page explains how to create a folder in **Assets** to organize asset files. It is intended for users managing assets within a space and should be used when setting up or maintaining a clean folder structure for easier browsing, filtering, and project organization.

### Item 1

#### Article section

##### Heading

Create a Folder

##### Content

Folders help you organize assets in **Assets**, making it easier to manage related files, group assets by projects, or maintain clean library structures. You can create folders directly from the Assets listing page.

To create a folder, perform the following steps:
- Navigate to the **Assets** listing page within your space.
- Click **+ New** and select **New Folder**.
- Enter the following details:**Name** (required)
- **Description** (optional)
- Click **Create**.

The created folder appears on the assets listing page and is also accessible under the **Folders** section in the **Filters** panel on the left.

**Tip:** Use clear, consistent folder names (e.g., Campaign Assets or Training Videos) to make searching and filtering easier later.

## Common questions

### Where do I create folders?
You can create folders directly from the **Assets** listing page within your space.

### Where can I find the folder after creating it?
The created folder appears on the assets listing page and is also accessible under the **Folders** section in the **Filters** panel on the left.

### Is the folder name required?
Yes, **Name** is required when creating a folder.

### Is the description required?
No, **Description** is optional.

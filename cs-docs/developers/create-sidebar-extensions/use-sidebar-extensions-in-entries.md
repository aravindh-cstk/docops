# [Extensions] - Use Sidebar Extensions in Entries

This page explains how to use a Sidebar Extension within entries for selected content types in a Contentstack stack. It is intended for developers and content managers who have already added a Sidebar Extension and need to run it from the entry sidebar while authoring content.

## Use Sidebar Extensions in Entries

**Note**: Experience Extensions use the legacy approach with extensions. We recommend using the  [Sidebar UI Location](../developer-hub/sidebar-location.md) for the Contentstack App Framework to extend the functionality of your apps.

Once you have added a [Sidebar Extension](./about-sidebar-extensions.md) in your [stack](../set-up-stack/about-stack.md), you can use it in the [entries](../../content-managers/author-content/about-entries.md) of selected [content type(s)](../create-content-types/about-content-types.md).

To use Sidebar Extensions in an entry, log in to your [Contentstack account](https://www.contentstack.com/login/), and perform the following steps:
- In your stack, click on the “Entries” icon on the left navigation panel.
- Select the content type for which you have enabled the Sidebar Extension.
- [Create an entry](../../content-managers/author-content/create-an-entry.md), click on the "Sidebar Extensions"** **icon located on the right panel. Select the Sidebar Extension that you want to add to the entry from the drop-down menu.
- Then, the selected Sidebar Extension will expand in the sidebar. Run it to see the results.

Once done using the Sidebar Extension, click on the close (x) button.

## Common questions

**How do I make a Sidebar Extension available for an entry?**  
Enable or add the [Sidebar Extension](./about-sidebar-extensions.md) in your [stack](../set-up-stack/about-stack.md) and ensure it is enabled for the selected [content type(s)](../create-content-types/about-content-types.md).

**Where do I access Sidebar Extensions while editing an entry?**  
Click on the “Entries” icon, open or [Create an entry](../../content-managers/author-content/create-an-entry.md), then click the "Sidebar Extensions" icon on the right panel.

**What should I use instead of legacy Experience Extensions?**  
Use the [Sidebar UI Location](../developer-hub/sidebar-location.md) for the Contentstack App Framework.

**How do I close a Sidebar Extension after using it?**  
Click on the close (x) button.

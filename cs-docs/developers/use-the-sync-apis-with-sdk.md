## Common questions

### What is the Sync API used for?
The Sync API takes care of syncing your Contentstack data with your app and ensures that the data is always up-to-date by providing delta updates.

### Where do I find SDK-specific instructions for using the Sync API?
Use the links in the “Guide on How to Sync APIs with SDKs” section to open the guide for your SDK (iOS, Android, JavaScript, Ruby, Dart, Python, Swift, PHP).

### Does this page contain implementation steps?
No, this is a listing page that provides an introduction and links to platform-specific guides.

### Which SDKs are covered on this page?
iOS, Android, JavaScript, Ruby, Dart, Python, Swift, and PHP.

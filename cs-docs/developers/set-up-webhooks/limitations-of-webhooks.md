# [Set Up Your Project] - Limitations of Webhooks

This page lists the platform limits for webhooks in a stack, including maximum counts for webhooks, notification email addresses, and webhook conditions. It is intended for developers and administrators configuring webhooks and planning webhook usage at scale.

## Limitations of Webhooks

- The maximum limit for creating webhooks in a stack is **100**. If you want to add more, please contact our [support team](mailto:support@contentstack.com).
- The maximum limit for the number of email addresses that can be notified whenever Webhook Circuit Breaker disables a specific webhook is **10**.
- The maximum number of conditions that can be added to a webhook is **200**.

## Common questions

**Q: What should I do if I need more than 100 webhooks in a stack?**  
A: Contact the [support team](mailto:support@contentstack.com).

**Q: How many email addresses can be notified when the Webhook Circuit Breaker disables a webhook?**  
A: The maximum limit is **10**.

**Q: How many conditions can I add to a webhook?**  
A: The maximum number of conditions is **200**.

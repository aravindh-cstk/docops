# [Marketplace guides and apps] - Content Intelligence App Installation Guide

This page explains how to install and configure the Content Intelligence Marketplace app in Contentstack and how to use it inside the JSON Rich Text Editor. It is intended for Contentstack Organization/Stack Owner/Admin users who want to enable auto suggestion, grammar correction, and spell correction while authoring entries.

Content Intelligence App Installation Guide

The Content Intelligence Marketplace app provides auto suggestion, grammar correction, and spell correction within the JSON RTE. You can easily use the Content Intelligence features to improve the accuracy and quality of your content in real-time.

Contentstack Marketplace allows you to readily install the Content Intelligence application to your stack. With the Content Intelligence button in the [JSON Rich Text Editor](../json-rich-text-editor.md), you can now enable services for auto suggestion, grammar correction, and spell correction within your Contentstack entry.

## Prerequisites

- [Contentstack account](https://app.contentstack.com/#!/login)
- Access to the Contentstack Organization/Stack as the Owner/Admin

This step-by-step guide explains how to install and configure the Content Intelligence app within your stack.

## Steps for Execution

- [Install and Configure Content Intelligence in Contentstack Marketplace](#install-and-configure-content-intelligence-in-contentstack-marketplace)
- [Use Content Intelligence within your Stack](#use-content-intelligence-within-your-stack)

## Install and Configure Content Intelligence in Contentstack Marketplace

Follow the steps below to install the Content Intelligence app in Contentstack.

Log in to your [Contentstack account](https://app.contentstack.com/#!/login).

- In the left-hand-side primary navigation, you will find a new icon for Marketplace (as shown below). Click the icon to go to the Marketplace.![MP-Apps.png](https://images.contentstack.io/v3/assets/blt23180bf2502c7444/bltc6ba36a89a14a721/63ca92b8a362596b80751544/MP-Apps.png)
- Within the Marketplace, you will be able to see all the available apps. Hover over the **Content Intelligence **app and click **Install App**.
- In the popup window, select the stack where you want the Content Intelligence app to be installed and click the **Install **button.![Content-Intelligence-Install-App.png](https://images.contentstack.io/v3/assets/blt23180bf2502c7444/blt57baf407740ec783/63ca92b79ba93014827b06c5/Content-Intelligence-Install-App.png)
- Click the **Save** button.
- Click **Open Stack** to start using the Content Intelligence application.

**Note: **No additional settings are required to use the Content Intelligence app.

## Use Content Intelligence within your Stack

To use the Content Intelligence application within an entry of your stack, follow the steps given below:

Go to your stack, click the “**Content Models**” icon in the left navigation panel, and click the **+ New Content Type** button.

- Create a content type by adding relevant details as displayed below:![Content-Intelligence-ContentType.png](https://images.contentstack.io/v3/assets/blt23180bf2502c7444/bltef213831128f82ab/63ca92b727d0de75938fc461/Content-Intelligence-ContentType.png)
- In the Content Type Builder page, add a JSON Rich Text Editor field in your content type by clicking the “**Insert a field**” link represented by a** + **sign.
- Under **Select JSON RTE Plugin(s)**, select **Content Intelligence**, and then click **Add Plugin(s)**.![Content-Intelligence-JSON-RTE-Added-Plugin.png](https://images.contentstack.io/v3/assets/blt23180bf2502c7444/blt338a91f6570bd165/63ca92b7dd49ff3d9fff24fa/Content-Intelligence-JSON-RTE-Added-Plugin.png)
- After adding the app in a JSON Rich Text Editor field, click either **Save** or **Save and Close** to save your changes.
- To use the Content Intelligence app, create an entry for this content type, and you will see the Content Intelligence button in the JSON Rich Text Editor field on your entry page as shown below:![Content-Intelligence-Sample-Entry.png](https://images.contentstack.io/v3/assets/blt23180bf2502c7444/bltbe667e2c6bf22e76/63ca92b7a1583f4c85f81d85/Content-Intelligence-Sample-Entry.png)
- The Content Intelligence app provides three features for a better writing experience. By default, all three features are enabled. You can choose to enable/disable each feature using the toggle buttons.

In the **JSON Rich Text Field**, enter the required text and to see how these tools work:**Auto Suggestion**: Provides relevant contextual suggestions to help complete a sentence by automatically predicting the next word or phrase.  
Enter a meaningful sentence. You will see that the Content Intelligence app provides contextual suggestions in gray color to complete the sentence. Press the tab key to accept the suggestion, or continue typing your preferred text if you do not want to use the suggestions.

- **Grammar Correction**: Detects and suggests grammar corrections when you enter a grammatically incorrect text.  
  Enter a grammatically incorrect sentence followed by a period. You will see the grammar error underlined in yellow color. Hover over the yellow-underlined text to find the grammar suggestion and then click the required suggestion to fix the grammar error.![Content-Intelligence-Grammar-Correction.png](https://images.contentstack.io/v3/assets/blt23180bf2502c7444/blt5c8150b45ea6c099/63ca92b75291cc643f1f1c3f/Content-Intelligence-Grammar-Correction.png)
- **Spell Correction**: Detects and suggests multiple spelling corrections when you enter a text with incorrect spelling.  
  Enter any text with a spelling error and press the spacebar key. You will see the error is underlined in red color. Hover over the red-underlined text to find the spelling suggestions and then click the appropriate suggestion to fix the spelling error.![Content-Intelligence-Spell-Correction.png](https://images.contentstack.io/v3/assets/blt23180bf2502c7444/blt52c6f1af3dfac381/63ca92b7adb01e2e7b052f4b/Content-Intelligence-Spell-Correction.png)
- After adding your content, **Save** and **Publish **your entry.

## Common questions

### Do I need to configure any additional settings after installing the app?
No. **Note: **No additional settings are required to use the Content Intelligence app.

### Where do I enable the Content Intelligence app for use in entries?
Under **Select JSON RTE Plugin(s)**, select **Content Intelligence**, and then click **Add Plugin(s)** in the JSON Rich Text Editor field configuration.

### What features does the Content Intelligence app provide in the JSON RTE?
It provides auto suggestion, grammar correction, and spell correction within the JSON RTE.

### Who needs access to install the Content Intelligence app?
Access to the Contentstack Organization/Stack as the Owner/Admin is required.

# [Automations guides and connectors] - Mark a Project as Favorite

This page explains how to pin a preferred project at the top by marking it as a favorite in Contentstack Automate. It is intended for developers and users managing projects in Automate, and should be used when you want quicker access to frequently used projects and to filter projects using tags and search.

## Mark a Project as Favorite

You can pin your preferred project at the top by marking it as your favorite project.

To mark a project as a favorite, follow the steps below:
- Log in to your [Contentstack account](https://www.contentstack.com/login/).
- In the left navigation panel, click **Automate**.
- Click the** star **icon present in the top right corner of your project card to mark it as favorite

You can also filter projects based on the tags.
These tags can help you filter out the projects in the Search. If you use both; a tag filter and the Project Search, then it gives a consolidated list of projects based on the search and tags.
- Refresh the page to view your favorite project(s) at the top.

## Common questions

**How do I make a project appear at the top of the projects list?**  
Mark the project as a favorite by clicking the star icon on the project card, then refresh the page.

**Where is the favorite (star) icon located?**  
It is present in the top right corner of your project card.

**Can I filter projects using both tags and search?**  
Yes. If you use both; a tag filter and the Project Search, then it gives a consolidated list of projects based on the search and tags.

**Do I need to refresh to see the updated favorites list?**  
Yes, refresh the page to view your favorite project(s) at the top.

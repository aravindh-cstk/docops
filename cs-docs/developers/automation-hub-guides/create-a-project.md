# [Automations guides and connectors] - Create a Project

This page explains how to create a project in Contentstack Automate to organize your automations. It is intended for developers and administrators setting up Automate for an organization and should be used when you need a new project container before building automations.

## Create a Project

A project in Automate, helps you organize all your automations in one location in an organized manner. To create a project, perform the steps given below:
- Log in to your [Contentstack account](https://app.contentstack.com/#!/login).
- In the left navigation panel, click the **Automate** icon.
- Click **+ New Project**.![Automate_-_Create_a_Project_-_Create_a_new_project](https://images.contentstack.io/v3/assets/blt2d43f51baca745a8/blt75cdf101c076e3a3/660301562531f477acee55b7/Automate_-_Get_Started_with_Automate_-_Create_a_new_project.png)
- In the **New project** model, enter the **Project Name** (for example, Sample-Automation), an optional **Description**, and click **Create**.![Create_Project.png](https://images.contentstack.io/v3/assets/blt2d43f51baca745a8/bltc28f7d22f184b889/659ab277c3fb279d9219ef64/Create_Project.png)

After successfully creating the project, you can start building different automations.

**Note: **The maximum number of projects allowed per organization is **50.**

## Common questions

### How many projects can I create per organization?
The maximum number of projects allowed per organization is **50.**

### Where do I create a new project in Automate?
In the left navigation panel, click the **Automate** icon, then click **+ New Project**.

### Is the project description required?
No. The **Description** is optional.

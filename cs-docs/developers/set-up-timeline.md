## Common questions

**What is this page used for?**  
This page provides a set of links to Timeline-related documentation, including setup guides, Preview API, and additional Timeline resources.

**Who should use Timeline?**  
Timeline is intended for content managers and developers who want to preview a site's appearance with scheduled updates and view upcoming content transformations.

**Where do I find the Preview API documentation?**  
In the "Work with Preview API" section under the link titled "Preview API" with Url: /developers/set-up-timeline/preview-api.

**Where can I learn about Timeline limitations or FAQs?**  
Use the "More about Timeline" section links: "Limitations for Timeline" and "Timeline FAQs".

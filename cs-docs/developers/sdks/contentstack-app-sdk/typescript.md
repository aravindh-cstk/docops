# Contentstack App SDK

This page introduces the Contentstack App SDK for TypeScript and provides navigation to key guides and reference material. Developers building custom apps that run within Contentstack should use this page to find the right starting point and supporting resources.

## Contentstack App SDK

The Contentstack App SDK lets developers build custom apps that run within Contentstack. It provides APIs for app initialization, UI location access, and interaction with entry, field, and app context data.

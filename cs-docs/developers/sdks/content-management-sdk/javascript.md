# [Second level navigation] - JavaScript Management SDK

This page is a listing hub for the JavaScript Management SDK documentation, intended for developers who want to fetch and/or manage Contentstack CMS data from JavaScript apps, and who need quickstart guidance, OAuth setup, references, and related SDK resources.

### Item 1

#### Listing page

##### Heading

JavaScript Management

##### Introduction

The JavaScript Management guide covers a series of topics on our CMS that ranges from a quickstart guide, queries that you can send to Contentstack to fetch and/or manage data for your JavaScript app, links to download the Delivery and Management SDKs for JavaScript, and sample apps to get you started.

A: It is a listing page that links to JavaScript Management SDK guides and resources, including getting started, OAuth 2.0, and the API reference.

A: Under “SDK Resources” → “JavaScript Management SDK API Reference” at `/developers/sdks/content-management-sdk/javascript/reference/`.

**Q: Is there guidance for OAuth 2.0 with the JavaScript Management SDK?**  
A: Yes, see “Implementing OAuth 2.0 with JavaScript Management SDK” at `/developers/sdks/content-management-sdk/javascript/implement-oauth-2-0-with-javascript-management-sdk`.

**Q: What topics does the JavaScript Management guide cover?**

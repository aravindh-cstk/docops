# [Contentstack Command-line Interface (CLI)] - Useful Plugins

This page lists useful plugins for the Contentstack Command-line Interface (CLI), intended for developers who want to extend or enhance their CLI workflows using plugins built on the Contentstack base class.

## Useful Plugins

Here are some plugins developed using the Contentstack base class that you can use. These plugins are either created by the Contentstack team or the Contentstack community.

- [contentstack-cli-tsgen](https://www.npmjs.com/package/contentstack-cli-tsgen) - Generate TypeScript typings from the content types

## Common questions

### What are these plugins built on?

These plugins are developed using the Contentstack base class.

### Who creates these plugins?

These plugins are either created by the Contentstack team or the Contentstack community.

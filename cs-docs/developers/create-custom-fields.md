# [Second level navigation] - Create Custom Fields

This page lists resources for creating and using Custom Fields, including prebuilt and external app custom field guides. It is intended for developers extending Contentstack functionality and should be used when you need to add or integrate custom fields in content types.

### Item 1

#### Listing page

##### Heading

Create Custom Fields

##### Introduction

**Note:** Experience Extensions use the legacy approach with extensions. We recommend using the [Custom Field UI location](./developer-hub/custom-field-location.md) for the Contentstack App Framework to extend the functionality of your apps.

The Custom Fields extension lets you add/create custom fields that you can use in your content type.

## Common questions

1. **Should I use Experience Extensions or the Custom Field UI location?**  
   Experience Extensions use the legacy approach with extensions. We recommend using the Custom Field UI location for the Contentstack App Framework.

2. **What can I do with the Custom Fields extension?**  
   The Custom Fields extension lets you add/create custom fields that you can use in your content type.

3. **Where can I find guides for prebuilt and external app custom fields?**  
   Use the links under “Prebuilt Custom Field guides” and “External Apps Custom Field guides” in this page.

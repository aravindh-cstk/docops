# [Integrations] - Recommended Zap Templates

This page lists recommended Zapier Zap templates for connecting Contentstack with popular third-party applications. It is intended for developers and integrators who want to quickly automate common workflows using Contentstack triggers and actions in Zapier.

## Recommended Zap Templates

Zapier lets you connect Contentstack with thousands of other applications, opening up endless possibilities.  
To help you get started with Contentstack and Zapier, we have created some Zap templates that can help you automate some basic workflows.  
Additionally, you can create custom Zaps by clicking on the **Make a Zap** button after logging in your Zapier account.

**Additional Resource: **To learn more about Contentstack's Zapier integration, please read our detailed guides on[integrating Zapier with Contentstack](./integrate-contentstack-with-other-apps-using-zapier.md)and learning to [build your own Zaps using Contentstack triggers and actions](./contentstack-triggers-and-actions-for-zapier.md).

Here are some recommended Zap templates you can use to connect Contentstack to other applications.

## Contentstack + Slack Zap Templates

Zapier allows you to connect Contentstack with Slack. This integration helps you receive notifications on Slack based on triggers that you specify in Contentstack.

- Zap template to send a **Slack notification** to a **channel **whenever a particular **workflow stage **is **updated** in Contentstack.  
  [Use the Zap Template](https://zapier.com/apps/contentstack/integrations/slack/765794/get-notified-on-slack-channels-when-workflow-stages-are-updated-in-contentstack)
- Zap template to send a **Slack notification**  to a **channel **when an **entry** is **published** in Contentstack.  
  [Use the Zap Template](https://zapier.com/apps/contentstack/integrations/slack/764471/relay-slack-channels-messages-when-new-entries-are-published-in-contentstack)

## Contentstack + Asana Zap Templates

You can connect Contentstack to your Asana account in order to perform various actions in Contentstack when trigger events occur in Asana, and vice versa.

- Zap template to **create an entry **in Contentstack whenever a **new task** is **created **in your Asana project.  
  [Use the Zap Template](https://zapier.com/apps/asana/integrations/contentstack/765797/create-entries-in-contentstack-from-new-tasks-in-asana-projects)
- Zap template to **update an entry **in Contentstack whenever a** task** is **updated **in your Asana project.  
  [Use the Zap Template](https://zapier.com/apps/asana/integrations/contentstack/765800/update-contentstack-entries-when-asana-tasks-are-newly-updated)
- Zap template to **create a task **in Asana whenever a **new entry** is **created **in Contentstack.  
  [Use the Zap Template](https://zapier.com/apps/asana/integrations/contentstack/765802/create-asana-tasks-from-new-contentstack-entries)

## Common questions

### Do I need to build a Zap from scratch to use these workflows?
No. You can use the provided links labeled **Use the Zap Template** to start from a prebuilt template.

### Where can I learn more about the Contentstack Zapier integration?
Use the **Additional Resource:** links for guides on integrating Zapier with Contentstack and learning to build your own Zaps using Contentstack triggers and actions.

### Can I create custom Zaps instead of using templates?
Yes. You can create custom Zaps by clicking on the **Make a Zap** button after logging in your Zapier account.

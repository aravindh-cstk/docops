# [Marketplace guides and apps] - Updating an Installed App

This page explains how to update an installed app in the Contentstack Marketplace. It is intended for developers and administrators who manage Marketplace apps in a stack and should be used when an installed app shows an available update.

## Updating an Installed App

To update an app, follow the steps below:
- Navigate to Contentstack Marketplace.
- Click **Manage** in the left-side navigation panel, and select **Installed Apps**.
- All apps with an available update are highlighted with a red dot.![Apps-With-Updates](https://images.contentstack.io/v3/assets/blt2d43f51baca745a8/blta5928b55589b003d/64de7939992f3c05a7869c1d/Apps-With-Updates.png)
- Click the app icon, then you can see a modal with details such as **Overview** (describes the app, its features and other information), **Screenshots** (shows few screenshots of the app), **Use Cases** (provides scenarios where an app can be used), and **Installed On** (includes the stack name, installed app version, who has installed the app, and the time of installation). By default, you are on the **Installed On** tab.

The red dot after the stack name signifies that an app update is available.
- Click **Update** to install the app updates.![Update-Icon-And-Button.png](https://images.contentstack.io/v3/assets/blt2d43f51baca745a8/blt435eaa0bd0e18e52/64de79391a23ce0b3f5b9cac/Update-Icon-And-Button.png)
- In the popup window, the stack is, by default, selected, accept the Terms of Service and Privacy Policy, and click the **Update** button.![Install-Updates](https://images.contentstack.io/v3/assets/blt2d43f51baca745a8/bltca2e11038045db1a/64de79394eb7009d2df7eb38/Install-Updates.png)
- Optionally, you can also update the existing configuration details which were configured when you first installed the app.![Configuration](https://images.contentstack.io/v3/assets/blt2d43f51baca745a8/blt201d7433ff09c173/64de7938adbf21436287d3df/Configuration.png)
- Click the **Save** button to save the updated configuration details.
- Click the **Open Stack** button to directly navigate to the stack.
- Click the **Cancel** button to cancel updating the configuration details.![Cancel-And-Open-Stack-And-Save-Buttons](https://images.contentstack.io/v3/assets/blt2d43f51baca745a8/blt959de12e572501b3/64de79395acb3b16d8301a94/Cancel-And-Open-Stack-And-Save-Buttons.png)

## Common questions

### How do I know an installed app has an update available?
All apps with an available update are highlighted with a red dot, and the red dot after the stack name signifies that an app update is available.

### Where do I find the update option for an installed app?
Click **Manage** in the left-side navigation panel, select **Installed Apps**, click the app icon, and then click **Update**.

### Can I change the app configuration while updating?
Optionally, you can also update the existing configuration details which were configured when you first installed the app, and then click the **Save** button to save the updated configuration details.

### What if I don’t want to proceed with updating configuration details?
Click the **Cancel** button to cancel updating the configuration details.

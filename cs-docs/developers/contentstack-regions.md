# Contentstack Regions

This page introduces Contentstack Regions and provides links to region-related documentation. It is intended for developers and administrators who need to understand where Contentstack data is hosted and how to choose and configure regions across APIs, SDKs, apps, and CLI.

## Contentstack Regions

A Contentstack region refers to the location of the data centers where your organization's data resides.

Contentstack currently supports seven regions: AWS North America, AWS Europe, AWS Australia, Azure North America, Azure Europe, GCP North America, and GCP Europe.

# [Set Up Your Project] - Limitations on Role Creation

This page describes the limitations on creating custom roles, including character limits and permission component limits. It is intended for developers setting up a project and assigning roles, and should be used when defining role titles, descriptions, and permissions.

## Limitations on Role Creation

- The number of characters allowed in a role title is within a range of **1** to **160**
- The maximum number of characters allowed in a role description is **320**
- The maximum allowed limit of components [Content Types, Entries, Assets, Locales, Environments] within the permissions of a custom role is **100**

## Common questions

### What happens if my role title is longer than 160 characters?
You will need to shorten the role title to be within the range of **1** to **160** characters.

### What is the maximum length for a role description?
The maximum number of characters allowed in a role description is **320**.

### How many permission components can a custom role include?
The maximum allowed limit of components [Content Types, Entries, Assets, Locales, Environments] within the permissions of a custom role is **100**.

# [Keyboard shortcuts] - App Keyboard Shortcuts

This page explains the available app keyboard shortcuts for quickly accessing Contentstack modules in the new user interface. It is intended for users who want faster navigation using Windows or Mac keyboard combinations, and should be used when learning or referencing module access shortcuts.

**Q: What are keyboard shortcuts used for in the app?**  
A: Keyboard shortcuts are keys or combinations of keys that provide you with an alternative way to perform an action.

**Q: Do these shortcuts work on both Windows and Mac?**  
A: The table lists shortcuts for both Windows OS and Mac OS where available.

**Q: Where can I find shortcuts for accessing specific modules?**  
A: Refer to the table under “App Keyboard Shortcuts,” which lists “Module to access” along with “Windows OS” and “Mac OS” shortcuts.

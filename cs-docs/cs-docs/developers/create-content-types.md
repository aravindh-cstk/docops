## Common questions

### What is this page used for?
It is a listing page that links to documentation pages related to creating, editing, and managing Content Types and their fields.

### Does this page include step-by-step instructions?
It primarily provides navigation links to detailed pages (for example, “Create a Content Type” and “Edit a Content Type”).

### Where can I find information about specific field types?
Use the “Fields” section links such as “Reference,” “Group,” “Modular Blocks,” or “JSON Rich Text Editor.”

### Where can I find limitations and FAQs?
See “Limitations of Content Types,” “Field Validation FAQs,” “Labels FAQs,” and “Content Type FAQs” in the “More about Content Types” and “Content Type Labels” sections.

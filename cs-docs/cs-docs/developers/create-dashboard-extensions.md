# [Second level navigation] - Create Dashboard Extensions

This page introduces Dashboard Extensions, a legacy extension approach for adding widgets to a stack dashboard. Developers extending app functionality should read this to understand Dashboard Extensions and when to use the recommended Dashboard UI Location in the Contentstack App Framework instead.

### Item 1

#### Listing page

##### Heading

Create Dashboard Extensions

##### Introduction

**Note**: Experience Extensions use the legacy approach with extensions. We recommend using the [Dashboard UI Location](/docs/developers/developer-hub/dashboard-location/) for the Contentstack App Framework to extend the functionality of your apps.

The Dashboard Extension is a type of extension that lets you create widgets for your stack dashboard.

## Common questions

**Q: Are Dashboard Extensions the recommended way to extend apps?**  
A: No. The page notes that Experience Extensions use the legacy approach with extensions and recommends using the Dashboard UI Location for the Contentstack App Framework.

**Q: What is a Dashboard Extension used for?**  
A: It lets you create widgets for your stack dashboard.

**Q: Where can I find examples of Dashboard Extensions?**  
A: The links include “Google Analytics Dashboard Extension” and “Content Type Visualizer Dashboard Extension.”

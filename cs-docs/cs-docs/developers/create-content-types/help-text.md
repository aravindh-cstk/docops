# [Set Up Your Project] - Help Text

This page explains the **Help Text** property available when configuring fields in a content type, who can use it, and how it appears to users when creating and viewing entries.

## Help Text

The **Help Text** property gives you the provision to add additional help text to a [field](./about-fields.md).

When you [create a content type](./create-a-content-type.md) and add a field to it, in the **Edit Properties** section, you will see this property.

After you configure a field with this property and [create an entry](../../content-managers/author-content/create-an-entry.md), then on the [entry](../../content-managers/author-content/about-entries.md) page you will see a help text icon (a question mark, "?") that appears beside the field name. ## Common questions

### Where do I find the Help Text property when setting up a field?
When you create a content type and add a field to it, in the **Edit Properties** section, you will see this property.

### What does the Help Text look like on an entry page?
You will see a help text icon (a question mark, "?") that appears beside the field name, and when the user hovers over this icon, the description appears as a tooltip.

### When will users see the Help Text tooltip?
After you configure a field with this property and create an entry, the tooltip appears on the entry page when hovering over the help text icon.

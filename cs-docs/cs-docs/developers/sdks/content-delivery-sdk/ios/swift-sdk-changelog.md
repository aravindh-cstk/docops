# [iOS] - Swift SDK Changelog

This page contains the release history for the iOS Swift SDK, including features, enhancements, and bug fixes by version. iOS developers should use it to understand what changed between SDK releases and to decide when to upgrade.

## Swift SDK Changelog

## Version 1.5.0 - May 26, 2023
**Feature:**
- Added support for Azure EU
- Added support for Include Metadata Implementation

**Bug Fixes:**
- Image Transform initialization bug

## Version 1.4.0 - Mar 9, 2023
**Enhancement:**
- `Entry` - Added support for `addValue : for HTTPHeaderField`
- `Asset `- Added support for `addValue : for HTTPHeaderField`
- `Query` - Added support for `addValue : for HTTPHeaderField`

## Version 1.3.1 - Nov 11, 2022
**Bug Fixes:**
- Resolved URL session cache default

## Version 1.3.0 - Aug 18, 2022
**Feature:**
- Added support for Azure EU region

## Version 1.2.6 - Aug 10, 2021
**Enhancement**
- Updated Utils SDK Package

## Version 1.2.5 - Jul 16, 2021
**Feature:**
- Added support for JSON RTE

## Version 1.2.4 - Apr 9, 2021
**Enhancement:**
- Added dependency for Cocoapods

## Version 1.2.3 - Apr 9, 2021
**Enhancement:**
- Added Utils SDK with function support

## Version 1.2.2 - Apr 6, 2021
**Bug Fixes:**
- Resolved issue for Cocoa pods

## Version 1.2.1 - Apr 6, 2021
**Bug Fixes:**
- Resolved Package issue

## Version 1.2.0 - Apr 6, 2021
**Feature:**
- Added support for Embedded Items

## Version 1.1.0 - Dec 5, 2020
**Enhancements**:
- `Asset` -  Added support for `includeFallback`
- `AssetLibrary` - Added support for `includeFallback`
- `Entry` - Added support for `includeFallback`
- `Query` - Added support for `includeFallback`

## Version 1.0.0 - May 11, 2020
Initial release

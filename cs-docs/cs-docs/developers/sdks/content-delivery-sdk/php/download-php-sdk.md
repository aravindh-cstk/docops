# [PHP] - Download PHP SDK

This page provides the download link for the Contentstack PHP SDK and a related resource link. PHP developers should use this page when setting up the SDK for PHP projects that fetch content via Contentstack Content Delivery APIs.

## Download PHP SDK

The Contentstack’s [PHP SDK](/docs/developers/php/about-php-sdk) interacts with and uses our [Content Delivery APIs](/docs/developers/apis/content-delivery-api) to fetch content from Contentstack. To use this SDK for your PHP projects, download it from the link below.

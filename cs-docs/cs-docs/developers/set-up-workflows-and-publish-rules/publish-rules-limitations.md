# [Set Up Your Project] - Publish Rules Limitations

This page lists the limitations for Publish Rules. It is intended for developers configuring workflows and publish rules, and should be used when validating constraints such as comment length.

## Publish Rules Limitations

- The maximum character length of comments is **200**

## Common questions

### What is the maximum character length of comments?
The maximum character length of comments is **200**.

### Where can I find Publish Rules limitations?
They are documented on the **Publish Rules Limitations** page under **Set Up Your Project**.

### What should I do if I need longer comments than 200 characters?
You must keep comments within **200** characters to comply with the Publish Rules limitation.

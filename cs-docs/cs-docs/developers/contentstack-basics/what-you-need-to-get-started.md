# [Contentstack Basics} - What You Need to Get Started

This page outlines the minimum requirements to run the Contentstack app at optimal performance. It is intended for developers and users setting up or validating their environment, and should be used when checking browser, OS, network, and accessibility compatibility before using Contentstack.

## What You Need to Get Started

The following sections outline the minimum requirements needed to run the Contentstack app at optimal performance.

## Supported Browsers and Operating Systems

Contentstack is compatible with all current browsers and operating systems. Typically, Contentstack only supports browser versions that were released within the last **2.5 years** and have a usage percentage of at least** 5%**.

The following list of browsers is optimized to work with Contentstack and its features:

## Screen Resolution

For best display results, use the system's recommended resolution for your monitor.

## Internet Connection

Use a high-speed internet connection to have an uninterrupted working experience with Contentstack.

## JavaScript and Cookie Requirements

Modern web apps depend on client-side features such as JavaScript and cookies. Therefore, for browser settings, ensure **JavaScript** is enabled and **cookies** are not blocked so that Contentstack can function correctly. Ad blockers or privacy extensions may interfere with certain features.

## Third-party Extensions or Add-ons

Some browser add-ons (such as script blockers, ad blockers, or privacy tools) may interfere with Contentstack features. Disable these if you encounter functionality issues.

## Accessibility Considerations

Contentstack follows modern web accessibility practices and works with most screen readers and assistive technologies. For best results, use the latest browser versions with accessibility support enabled.

## Network Requirements (Advanced)

- **Minimum recommended bandwidth**: **5 **Mbps
- **Ports**: Ensure HTTPS (port **443**) is open and unfiltered
- **VPN/Proxy**: Ensure Contentstack endpoints aren’t blocked

## Compatibility Notice for Deprecated Systems

**Note on Legacy Systems:** While Contentstack may load on Windows **7**/**8**/**8.1**, official support for these platforms is limited as they are no longer maintained by Microsoft.

## Common questions

### What browsers does Contentstack support?
Contentstack is compatible with all current browsers and operating systems, and typically supports browser versions released within the last **2.5 years** with a usage percentage of at least** 5%**.

### What network settings are required for Contentstack?
Ensure HTTPS (port **443**) is open and unfiltered, and that Contentstack endpoints aren’t blocked by a VPN or proxy.

### Do I need to enable JavaScript and cookies?
Yes—ensure **JavaScript** is enabled and **cookies** are not blocked so that Contentstack can function correctly.

### Can browser extensions affect Contentstack?
Yes—some add-ons (such as script blockers, ad blockers, or privacy tools) may interfere with Contentstack features and should be disabled if you encounter issues.

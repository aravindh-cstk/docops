# [Automations guides and connectors] - Rename Trigger

This page explains how to rename a trigger within an automation. It is intended for developers and automation builders who need to update trigger names for clarity or organization while configuring automations.

## Rename Trigger

To rename a trigger from the automation, perform the steps given below:
- Click the trigger tab (in our case, **HTTP Request Trigger**).
- Click the edit icon visible on the trigger header to **Rename Trigger**.
- The **HTTP Request Trigger** field becomes editable. Update the name as required and click the “✔” check mark.

## Common questions

### Can I rename any trigger in an automation?
Yes, you can rename a trigger from the automation by editing the trigger header name field.

### Does renaming a trigger change its configuration?
No. Renaming updates only the trigger name displayed in the automation.

### What should I do if I don’t see the edit icon?
Ensure you are viewing the trigger tab and have permission to edit the automation.

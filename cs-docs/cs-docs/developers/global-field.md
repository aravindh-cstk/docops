## Common questions

### What is a Global field used for?
A Global field is a reusable field (or group of fields) that you can define once and reuse in any content type within your stack.

### Where can I find instructions to create or edit a Global field?
Use the links under “Work with Global Field,” such as “Create a Global field” and “Edit a Global Field.”

### Where are advanced/complex Global Field topics documented?
See the “Complex Global Fields” section for topics like group fields, modular blocks, and reference fields within Global Fields.

### Where can I find limitations and FAQs for Global Field?
Check “More about Global Field,” which includes “Limitations of Global Field” and “Global Field FAQs.”

## Common questions

### What is Static Site Generation (SSG)?
Static Site Generation (SSG) generates website pages at build time. The content on web pages generated using SSG can be accessed quickly.

### Where can I find Contentstack plugins for static site generators?
Use the links provided in the Gatsby and Metalsmith sections, such as “Contentstack Gatsby Plugin” and “Contentstack Metalsmith Plugin.”

### Are there sample apps for getting started with these SSGs?
Yes, this page links to sample apps like “Get Started with Gatsby and Contentstack,” “Build a Website Using Metalsmith and Contentstack,” and “Build a Starter Website Using Next.js Static Site Generator and Contentstack.”

### Does this page include setup instructions?
No, it provides a listing of links to related pages and repositories where setup and implementation details are available.

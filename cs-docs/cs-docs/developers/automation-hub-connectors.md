# [Second level navigation] - Automate Connectors

This page lists Automate connectors (third-party applications) that can be used with Contentstack Automate to automate tasks. It is intended for developers and automation builders who need to discover available triggers and actions and navigate to the relevant connector documentation when configuring automations.

## Listing page

### Automate Connectors

Automate connectors are third-party applications that can be utilized with [Contentstack Automate](/docs/developers/automation-hub-guides/about-automation-hub) to automate specific tasks. Here are some connectors that you can swiftly configure and utilize for automation purposes.

## Common questions

### What are Automate connectors used for?
Automate connectors are third-party applications that can be utilized with Contentstack Automate to automate specific tasks.

### Where do I find triggers for starting an automation?
Use the links under the **Triggers** section, such as **Contentstack Trigger**, **HTTP Trigger**, or **Scheduler by Automate**.

### Where do I find Contentstack Management actions?
Use the links under **Contentstack Management**, such as **Contentstack Management - Entries Actions** or **Contentstack Management - Assets Actions**.

### How do I open the documentation for a specific connector?
Select the connector name in the relevant section to navigate to its URL (for example, **Slack** or **AWS Lambda**).

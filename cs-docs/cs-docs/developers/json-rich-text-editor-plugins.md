# [Second level navigation] - JSON Rich Text Editor Plugins

This page provides a listing of resources and guides related to JSON Rich Text Editor (JSON RTE) Plugins, including getting started links, prebuilt plugin guides, and usage guides. It is intended for developers extending JSON RTE functionality and should be used when navigating to plugin documentation and related references.

#### Listing page

##### Heading

JSON Rich Text Editor Plugins

##### Introduction

**Note**: Experience Extensions use the legacy approach with extensions. We recommend using the [RTE UI Location](/docs/developers/developer-hub/rte-location/) for the Contentstack App Framework to extend the functionality of your apps.

The JSON Rich Text Editor Plugins extension lets you add/create customized plugins to extend the functionality of your JSON RTE.

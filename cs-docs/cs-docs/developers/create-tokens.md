# [Second level navigation] - Create Tokens

This page is a listing/landing page for token-related documentation in Contentstack. Developers should use it to understand what tokens are used for and to navigate to specific guides for Access tokens, Delivery tokens, Management tokens, and Authtokens.

## Listing page

### Heading

Create Tokens

### Introduction

Tokens are needed to authorize API calls. There are following types of tokens in Contentstack: Access token, Delivery token, Management token, and Authtoken.

# [Contentstack Launch] - Delete a Project

This page explains how to delete a project in Contentstack Launch. It is intended for developers or team members managing Launch projects and should be used when you no longer require a project and want to remove it.

## Delete a Project

Launch allows you to delete a project which you no longer require.

This document guides you through the process of deleting a project in Launch.
- Select the project you want to delete.
- Click the **Settings** icon from the left panel.
- Click **Advanced**.

    You will see a warning message displayed above the delete button.
- Click the **Delete **button.

    A modal appears asking for confirmation to delete the current project.
- Enter the project name and click the **Delete **button.

You have successfully deleted your project.

## Common questions

### Can I undo deleting a project?
No. This page describes deleting a project, which removes it; ensure you no longer require the project before proceeding.

### Why do I need to enter the project name before deleting?
The confirmation step helps ensure you are deleting the correct project intentionally.

### Where do I find the delete option for a project?
In the project, click the **Settings** icon from the left panel, then click **Advanced** to access the delete option.

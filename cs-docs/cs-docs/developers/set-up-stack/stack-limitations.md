# [Set Up Stack] - Stack Limitations

This page lists the limitations for Stack. It is intended for developers who are setting up and managing stacks and should be used when planning stack creation frequency, branch usage, and environment configuration.

## Limitations for Stack

- You can create only **one stack per minute**.
- Each stack can have a maximum of **2 branches**.
- Each stack supports only **3 environments**.

## Common questions

### What happens if I try to create more than one stack per minute?

You can create only **one stack per minute**.

### How many branches can a stack have?

Each stack can have a maximum of **2 branches**.

### How many environments does each stack support?

Each stack supports only **3 environments**.

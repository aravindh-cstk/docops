# [Author Content] - Bulk Delete Assets

This page explains how to use Contentstack’s Bulk Delete Assets feature to remove multiple assets from a stack. It is intended for content managers and developers who manage assets in Contentstack and need to delete assets in batches and understand trash/retention behavior.

## Bulk Delete Assets

Contentstack’s Bulk Delete Assets feature provides instructions for removing multiple assets from your stack.

To bulk delete assets, log in to your [Contentstack account](https://www.contentstack.com/login), and perform the following steps:
- Go to your [stack](/docs/developers/set-up-stack/about-stack) and click the “Assets” icon in the left navigation panel. You can use the shortcut key “A” (for both Windows and Mac OS).
- Use the checkboxes to select the assets you want to delete.
- After selecting the assets, click the **Delete** option in the floating panel that appears.
- A confirmation dialog will appear. Click Delete to move the selected assets to the [Trash](/docs/developers/manage-trash/about-trash).**Note: **You can select and delete up to **100** **assets** at a time. To delete more than **100** **entries**, repeat the process in batches.

**Note:** Deleted assets are moved to the trash and can be restored within **14** **days**. After this period, they will be permanently deleted.

## Common questions

### How many assets can I bulk delete at a time?
You can select and delete up to **100** **assets** at a time.

### What happens after I delete assets in bulk?
Deleted assets are moved to the [Trash](/docs/developers/manage-trash/about-trash).

### Can I restore deleted assets?
Yes. Deleted assets can be restored within **14** **days**.

### How do I delete more than 100 assets?
Repeat the process in batches to delete more than **100** items.

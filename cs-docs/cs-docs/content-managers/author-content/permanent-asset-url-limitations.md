# [Author Content] - Permanent Asset URL Limitations

This page describes the limitations of permanent asset URLs in Author Content, intended for content managers who generate permanent URLs for assets and need to understand slug length constraints and update restrictions.

## Permanent Asset URL Limitations

- The slug for a permanent asset URL can have a maximum length of **255 characters**.
- You can only generate the permanent URL and update asset details once. After this, the permanent URL cannot be changed.

## Common questions

### What is the maximum slug length for a permanent asset URL?
The slug for a permanent asset URL can have a maximum length of **255 characters**.

### Can I change a permanent asset URL after generating it?
No. You can only generate the permanent URL and update asset details once. After this, the permanent URL cannot be changed.

### How many times can I update asset details when generating a permanent URL?
You can only generate the permanent URL and update asset details once.

# [Author Content | Entry Versions] - Compare Entry Versions

This page explains how to compare two versions of a Contentstack entry side-by-side (including across languages) to identify differences using a color-coded comparison view. It is intended for content managers and editors who need to review changes, track revisions, or validate translations across entry versions.

## Compare Entry Versions

Contentstack enables you to compare any two versions of an entry side-by-side on the same screen. With a color-coded comparison view, you can easily identify differences between the selected entry versions.

You can also compare versions of the same entry in different languages. For instance, compare the version 4 in **English** with version 2 in **German** to spot differences across translations.

To compare versions of an entry, log in to your [Contentstack account](https://www.contentstack.com/login), and perform the following steps:
- Go to your [stack](/docs/developers/set-up-stack/about-stack) and click the “Entries”.
- Navigate to the entry you want to compare.
- Click the version icon in the top-right corner of the page or the “Versions” icon in the right sidebar.
- Locate the entry you want to compare with the current selected version. Hover over it and click the “Compare” icon.
- The comparison view will display the two selected versions side-by-side. Changes will be highlighted using color coding, making it easy to identify differences.
    If needed, you can switch to a different version or select another language for comparison by using the dropdown menus in the comparison view.

The ability to compare entry versions side-by-side ensures transparency and simplifies version management in Contentstack. By utilizing the color-coded comparison view and multi-language support, you can effortlessly review and track changes, ensuring your content stays accurate and consistent across versions and languages.

## Common questions

**How do I open the version comparison view for an entry?**  
Click the version icon in the top-right corner of the entry page or the “Versions” icon in the right sidebar, then hover over a version and click the “Compare” icon.

**Can I compare entry versions across different languages?**  
Yes, you can compare versions of the same entry in different languages (for example, version 4 in **English** with version 2 in **German**).

**How are differences shown in the comparison view?**  
Changes are highlighted using color coding in a side-by-side comparison view.

**Can I change which versions or languages are being compared after opening the view?**  
Yes, you can switch to a different version or select another language for comparison by using the dropdown menus in the comparison view.

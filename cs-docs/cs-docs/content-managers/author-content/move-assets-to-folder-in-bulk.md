# [Author Content] - Move Assets to Folder in Bulk

This page explains how Contentstack content managers and authors can use the Bulk Move feature to move multiple assets into a selected folder at once, and should be used when you need to reorganize assets efficiently without moving them one by one.

## Move Assets to Folder in Bulk

The Bulk Move feature in Contentstack allows you to efficiently transfer multiple assets to designated folders simultaneously. This feature reduces manual effort, ensures consistent categorization, and enhances content organization.

To move multiple assets to a folder, log in to your [Contentstack account](https://www.contentstack.com/login), and perform the following steps:
- Go to your [stack](/docs/developers/set-up-stack/about-stack) and click the “Assets” icon in the left navigation panel. You can also use the shortcut key “A” (for both Windows and Mac OS users).
- Use the checkboxes to select the assets you want to move.
- After selecting the assets, click the **Move** **To** option in the floating panel that appears.
- In the **Move** **To** modal, select the target folder where you want to relocate the assets. If needed, click the **+ New Folder** icon to create a new folder and move the assets into it.

Finally, click **Move** **here** to move the selected assets

## Common questions

### Can I create a new folder while moving assets in bulk?
Yes. In the **Move** **To** modal, click the **+ New Folder** icon to create a new folder and move the assets into it.

### Is there a keyboard shortcut to open Assets?
Yes. You can use the shortcut key “A” (for both Windows and Mac OS users).

### Where do I find the Bulk Move option?
After selecting the assets using the checkboxes, click the **Move** **To** option in the floating panel that appears.

# [Author Content] - Limitations on Asset Folders

This page lists the limitations on asset folders in Contentstack, including the maximum number of asset folders per stack and the allowed nesting depth. It is intended for content managers and authors who organize assets and need to understand or request changes to these limits.

## Limitations on Asset Folders

- You can create up to **10000 asset folders** per stack.
- A maximum of **5 levels** of asset folder nesting is allowed.

**Note**: If you need to increase these limits, please contact our [support](mailto:support@contentstack.com) team.

# [Author Content] - Bulk Publish Assets

This page explains how to publish multiple assets at once in Contentstack. It is intended for content managers and authors who need to publish images, documents, or other media files in bulk across environments and languages.

## Bulk Publish Assets

Publishing multiple assets simultaneously streamlines your workflow, saving time and effort compared to publishing assets individually. Whether you are handling images, documents, or other media files, this feature helps to optimize your publishing workflow in Contentstack.

To publish multiple assets at once, log in to your [Contentstack account,](https://www.contentstack.com/login/) and perform the following steps:

- Go to your [stack](/docs/developers/set-up-stack/about-stack) and click the “Assets” icon in the left navigation panel. You can also use the shortcut key “A” (for both Windows and Mac OS users).
- Use the checkboxes to select the assets you want to publish.
- After selecting the assets, click the **Publish** option in the floating panel that appears.
- In the **Publish** **Assets** modal:
  - **Select Environment(s):** Select the environment(s) where the assets will be published.
  - **Select Language(s):** Select the locale(s) in which you want the assets to be published.
  - **Note:** When publishing multiple assets, only their latest versions will be published.
- Click **Publish** to publish the selected assets.

**Note: **You can publish up to **10** **assets** simultaneously in **10** **languages** across **10** **environments**.

**Additional Resource:** For automated bulk publishing, refer to the [Bulk Publishing of Entries and Assets](/docs/developers/utilities/bulk-publishing-of-entries-and-assets) guide.

## Common questions

### How many assets can I publish at once?
You can publish up to **10** **assets** simultaneously.

### Can I publish the same assets to multiple environments and languages?
Yes. You can publish in **10** **languages** across **10** **environments**.

### Which versions of assets are published during bulk publishing?
When publishing multiple assets, only their latest versions will be published.

### Where can I find guidance for automated bulk publishing?
Refer to the **Additional Resource** link: [Bulk Publishing of Entries and Assets](/docs/developers/utilities/bulk-publishing-of-entries-and-assets).

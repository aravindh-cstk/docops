# [Author Content] - Limitations for Entries

This page lists the limitations for Author Content entries. It is intended for content managers and developers who need to understand constraints (such as reference query limits, tag limits, and publishing restrictions) when creating, managing, or publishing entries.

## Limitations for Entries

- The maximum query limit for the **Reference** field is **100**.
- An entry can have up to **50 tags**.
- Each tag can contain up to **50 characters**.
- Entries that have not yet been saved (**version 0**) cannot be published, added to releases, scheduled for publishing, included in bulk publishing, or used as references until the first version is created.

## Common questions

**What happens if I try to publish an entry that is version 0?**  
Entries that have not yet been saved (**version 0**) cannot be published until the first version is created.

**How many tags can I add to a single entry?**  
An entry can have up to **50 tags**.

**What is the maximum length of a tag?**  
Each tag can contain up to **50 characters**.

**What is the maximum query limit for the Reference field?**  
The maximum query limit for the **Reference** field is **100**.

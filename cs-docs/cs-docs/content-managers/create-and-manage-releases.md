This page lists documentation for creating and managing Releases, including related tasks like adding items, locking/unlocking, updating items, deploying, and managing limitations/FAQs. Content managers should use this page to navigate to the specific Release workflow they need.

## Create and Manage Releases

A "Release" is a set of entries and assets that needs to be deployed (published or unpublished) all at once to a particular environment.

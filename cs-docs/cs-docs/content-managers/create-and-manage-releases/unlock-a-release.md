# [Create and Manage Releases NEW] - Unlock a Release

This page explains how to unlock a release in Contentstack so you can modify the items within it. It is intended for Contentstack users managing releases and should be used when you need to make changes to a release that is currently locked.

## Unlock a Release

Unlocking a release enables you to make modifications to the items of a release.

To unlock a release, log in to your [Contentstack account](https://www.contentstack.com/login) and follow these steps:
- Go to your [stack](/docs/developers/set-up-stack/about-stack) and click the “Releases” icon in the left navigation panel.
- Hover over the release you want to unlock in the left panel.
- Click the “Unlock” icon in the popup modal. Alternatively, click the horizontal ellipsis in the top right corner and click **Unlock**.
- In the resulting **Unlock Release** modal, click the **Yes, Proceed** button to unlock your release.

## Common questions

**How do I know if a release is locked?**  
If you see an “Unlock” option/icon available for the release, it indicates the release is currently locked and can be unlocked.

**What changes after I unlock a release?**  
Unlocking a release enables you to make modifications to the items of a release.

**Do I need special access to unlock a release?**  
You must be able to log in to your Contentstack account and access the stack and Releases area to perform the unlock action.

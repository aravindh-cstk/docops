## Common questions

**How are asset versions identified by default?**  
By default asset versions are identified with numbers (for e.g., Version 1, Version 2).

**Can I assign a custom name to an asset version?**  
Yes, you can assign a name (for e.g., Production Ready, Do Not Edit) to an asset version for easy identification.

# [Visual Editor] - About Visual Editor

This page explains what the Visual Editor is, what it enables, and the key features and benefits it provides. It is intended for content creators, marketers, content managers, and developers who need to understand when and how to use Visual Editor for creating and managing web pages with live preview, workflows, and publishing.

## About Visual Editor

The Visual Editor provides you with a no-code solution for creating and managing web pages. It enables content creators and marketers to update content within a live preview of the site, helping reduce reliance on developers for routine changes.

With Visual Editor, you can modify page layouts, rearrange components, and create new content without technical skills. This WYSIWYG (What You See Is What You Get) interface ensured alignment Editor content creation with its final presentation, making it easier to review and publish changes quickly.

Visual Editor also supports real-time collaboration between marketers and developers, helping teams work more efficiently and maintain consistency across updates.

## Key Features and Benefits

Visual Editor offers several key capabilities that enhance the content creation and management experience:
- **Customizable Interface**: Design and update pages by dragging and dropping components, content blocks, or field modifiers directly on the canvas.
- **Real-Time Preview**: See instant feedback on edits in a live-site context, making it easier to identify and resolve issues before publishing.
- **Automated Content Delivery**: Set up event-based workflows that trigger automated actions across integrated applications.
- **Audience-Specific Content Editing**: Customize and preview content variations for different user segments, such as regions, languages, or customer types.
- **Workflow Management**: Simplify editorial processes with customizable workflows, in-context workflow transitions, and flexible publishing options like quick or advanced publishing for single or multiple [entries](/docs/content-managers/author-content/about-entries).
- **Multi-Channel Publishing**: Choose target environments such as staging or production during publishing to streamline content deployment across channels.

Visual Editor makes it easy to create and manage content with multi-entry editing, real-time validation, on-canvas field control, and smooth workflow and publishing tools.

# [Visual Editor] - Add to Release in Visual Editor

This page explains how to add entries to a release directly within Contentstack Visual Editor. It is intended for content managers and editors who prepare and organize content for publishing and should be used when you want to manage releases without leaving the page editing workflow.

### Item 1

#### Article section

##### Heading

Add to Release in Visual Editor

##### Content

[Visual Editor](/docs/content-managers/visual-editor/about-visual-editor) allows you to add one or more entries to a release directly from the page editing interface. This reduces context switching and helps content teams prepare and organize content for publishing without leaving the Visual Editor. Whether you are preparing a single update or coordinating changes across multiple entries, this feature simplifies release management within the editing workflow.

## Add Entries to a Release from Visual Editor
To add multiple entries to a release using Visual Editor, log in to your [Contentstack account](https://www.contentstack.com/login/) and perform the following steps:
- Navigate to your [stack](/docs/developers/set-up-stack/about-stack) and select **Visual Experience**.
- Click **Editor** in the bottom pill menu.
- Click the horizontal ellipsis and select **Add to Release**.
- In the **Add to Release** modal, select one or more entries from the list.A green checkmark indicates the entry is ready to be added.
- Entries without a green check may require review or may not meet publishing criteria.
- Click **Proceed**.
- Select an existing **Release** or create a new one.
- Select one or more **Locales**.
- Click **Add with Reference** to publish the referenced entries along with the selected entry.**Note:** You must select at least one entry to proceed.

## Add a Specific Entry to a Release
You can also add an individual entry to a release directly from the **Form panel** while viewing the entry in Visual Editor.

To add a specific entry to a release:
- Use the **URL bar** to open the page associated with the entry.
- Open the entry in the **Form panel**.
- Click the **entry actions** menu and select **Add to Release**.
- Choose the target **Release** and **Locales**.
- Click **Add** to confirm.

After the entries are added, a confirmation message appears indicating that the request is in progress. You can click **View Release** to monitor the status and track the entries included in the release.

## Common questions

**Q: Can I add multiple entries to a release from Visual Editor?**  
A: Yes, you can add one or more entries using the **Add to Release** option and selecting entries in the **Add to Release** modal.

**Q: What does the green checkmark mean in the Add to Release modal?**  
A: A green checkmark indicates the entry is ready to be added.

**Q: Can I add a single entry to a release without using the modal list?**  
A: Yes, you can add an individual entry from the **Form panel** using the **entry actions** menu and selecting **Add to Release**.

**Q: How do I track the status after adding entries to a release?**  
A: After adding entries, you can click **View Release** to monitor the status and track the entries included in the release.

# [Preview Sharing] - Limitations for Preview Sharing

This page lists the limitations for the Preview Sharing feature in Contentstack. It is intended for content managers and reviewers who use preview links to review content and leave comments, and should be referenced when setting expectations around access, commenting, and link behavior.

## Limitations for Preview Sharing

- The Preview Sharing feature is only available if your organization has the Live Preview and Visual Builder plan enabled. Please contact the [support](mailto:support@contentstack.com) team to activate this plan for your organization.
- Reviewers must receive a preview link to add comments.
- Expired preview links cannot be restored; a new link must be generated.
- External reviewers cannot edit the content, only comment on it.
- The maximum message length allowed for each comment is **500 characters**.
- You can mention a maximum of **20 users** and **20 roles** in a single comment.

## Common questions

**How do I choose which bulk operation to use?**  
Use the link that matches your intended action: publish, unpublish, delete, or change workflow details.

**Where can I learn what bulk operations do before running them?**  
Go to **About Bulk Operations on Search Results**.

**Does this page contain the steps to run bulk operations?**  
No. Use the links on this page to open the specific bulk operation documentation.

**What kinds of items can be affected by bulk operations from search results?**  
See **Publish Entries and Assets in Bulk**, **Unpublish Entries and Assets in Bulk**, and **Delete Entries and Assets in Bulk** for entries and assets, and **Change Workflow Details of Entries in Bulk** for workflow changes.

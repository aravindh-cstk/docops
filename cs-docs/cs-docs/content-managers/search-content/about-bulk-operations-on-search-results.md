# [Search Content] - About Bulk Operations on Search Results

This page explains how bulk operations work on search results in Contentstack and is intended for content managers who need to take actions (such as publishing or deleting) on multiple items at once after narrowing results using search, filters, and views.

## About Bulk Operations on Search Results

Contentstack allows you to perform bulk operations on search results, enabling [content managers](../../developers/invite-users-and-assign-roles/types-of-roles.md#content-manager) to select multiple items and perform actions such as **Publish**, **Unpublish**, **Delete**, or **Change Workflow Details** in bulk.

These bulk operations, integrated into the search functionality, lets you quickly locate content and take immediate action on selected items. You can use [Basic Search](./basic-search.md), [Advanced Search](./advanced-search.md), [Filters](./use-filters.md), and [Views](./about-views.md) to narrow down your search results before applying bulk operations to the refined list.

For example, if you want to publish all entries with the content type "Blog," search for "Blog" using Basic or Advanced Search. Apply relevant filters if needed, and save the search as a view for future use. From the search results, select the relevant entries and click the Publish option to bulk publish them.

**Additional Resource:** To learn how to perform bulk operations using the [Content Management API (CMA)](../../../api-docs/api-detail/content-management-api.md), refer to the [Bulk Operations](../../../api-docs/api-detail/content-management-api.md#bulk-operations) section.

## Common questions

### Who can perform bulk operations on search results?
[content managers](../../developers/invite-users-and-assign-roles/types-of-roles.md#content-manager) can select multiple items in search results and perform actions such as **Publish**, **Unpublish**, **Delete**, or **Change Workflow Details** in bulk.

### What actions are available as bulk operations from search results?
You can perform actions such as **Publish**, **Unpublish**, **Delete**, or **Change Workflow Details** in bulk.

### How do I narrow down items before running a bulk operation?
You can use [Basic Search](./basic-search.md), [Advanced Search](./advanced-search.md), [Filters](./use-filters.md), and [Views](./about-views.md) to narrow down your search results before applying bulk operations.

### Where can I learn about bulk operations via API?
Refer to the [Content Management API (CMA)](../../../api-docs/api-detail/content-management-api.md) documentation and the [Bulk Operations](../../../api-docs/api-detail/content-management-api.md#bulk-operations) section.

# Preview Sharing

This page introduces Preview Sharing and provides navigation to key tasks and reference topics for content managers who need to generate and share live preview links for review and approval without requiring stakeholder login or platform access.

### Introduction

Streamline your content reviews with secure, shareable previews. Preview Sharing lets you generate live links to in-progress websites, so stakeholders can view and approve content—no login or platform access needed. Fast, simple, and built for collaboration.

## Common questions

### What is Preview Sharing used for?
Preview Sharing lets you generate live links to in-progress websites so stakeholders can view and approve content—no login or platform access needed.

### Where do I start if I want to share a link for review?
Use the **Share a Preview** link in the **Work with Preview Sharing** section.

### How do I collaborate on feedback during a preview?
Use the review comment topics: **Add a Review Comment**, **Reply to a Review Comment**, **Resolve a Review Comment**, and **Reopen a Review Comment**.

### Where can I find constraints or known restrictions?
See **Limitations for Preview Sharing** under **More about Preview Sharing**.

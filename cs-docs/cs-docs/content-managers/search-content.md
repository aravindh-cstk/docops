# [Second level navigation] - Search Content

This page provides the Search Content section overview and link structure for Contentstack content managers. Read this when you need to understand or navigate to Search-related topics such as Basic/Advanced Search, Views, localization operator usage, bulk operations on search results, and search limitations/FAQs.

## Listing page

### Heading

Search Content

### Introduction

Contentstack's Search functionality allows you to efficiently find assets or entries within a specific stack. You can choose between Basic and Advanced Search options to tailor your search based on your needs, allowing for quick and precise results across all content.

## Common questions

### What is the purpose of the Search Content section?
It groups documentation links related to finding assets or entries in a stack using Basic and Advanced Search, filters, and quick search.

### Where can I learn about Advanced Search and supported operators?
Use the links labeled “Advanced Search” and “Supported Operators for Various Data Types” in the “Working with Search” section.

### How do I find documentation about Views (including shared or saved views)?
Go to the “Working with Views” section and open “About Views,” “Save Your Views,” “Shared Views,” or “Use Saved Views.”

### Where are bulk actions on search results documented?
See the “Bulk Operations on Search Results” section for links covering publish, unpublish, delete, and workflow changes in bulk.

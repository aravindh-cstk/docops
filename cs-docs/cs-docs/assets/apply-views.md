# [AM2.0] - Apply Views

This page explains how to use **Views** in Assets to organize and refine repetitive search results, including how to apply predefined or custom views, save new views, and manage saved views. It is intended for users working in the Assets section who need to quickly switch between saved search configurations and collaborate across teams.

## Apply Views

In Assets, **Views** help you organize and refine your repetitive search results. You can apply predefined or custom views, save new ones, and share them for improved cross-team collaboration.

Views let you quickly switch between saved search configurations:
- **Popular Views**: Predefined options such as **All Assets**, **Last Modified by Me**, **Published by Me**, or **Not Published**.
- **Saved Views**: Custom views you create are managed.

To apply a view:
- Navigate to your space.
- Go to the **Assets** section.
- Select the view you need. The asset list updates instantly.

## Save New Views

After applying filters or building a search, you can save the configuration as a new view:
- In the **Assets** section of your space, apply filters or advanced search.
- Open the current view dropdown from the top-right corner and choose **Save as New View**.
- Enter a name for the view and save it.

## Manage Saved Views

For any saved view, click the vertical ellipsis next to it and choose from the following options:
- **Rename**: Change the view name.
- **Share**: Share the view with users or roles in your stack.
- **Copy Link**: Share the view link with collaborators for quick access.**Note**: Anyone with the link can access the view and save it as their new view.
- **View Details**: See key information such as creator, last modified date, and access permissions.
- **Delete**: Permanently remove the view if no longer needed. For shared views, the owner can remove it for all users or transfer ownership to another user before deleting it from their account.

With saved views, you can quickly switch between different search scenarios, saving your time and effort.

## Common questions

**How do I apply a view in Assets?**  
Navigate to your space, go to the **Assets** section, and select the view you need; the asset list updates instantly.

**How do I save a new view?**  
In the **Assets** section, apply filters or advanced search, open the current view dropdown from the top-right corner, choose **Save as New View**, enter a name, and save it.

**What can I do from the saved view vertical ellipsis menu?**  
You can **Rename**, **Share**, **Copy Link**, view **View Details**, or **Delete** the saved view.

**Who can delete a shared view?**  
For shared views, the owner can remove it for all users or transfer ownership to another user before deleting it from their account.

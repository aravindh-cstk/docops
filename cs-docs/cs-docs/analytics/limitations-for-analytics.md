# [Analytics Content] - Limitations for Analytics

This page lists the current limitations for Analytics dashboards and filtering, and is intended for developers and analysts who need to understand data freshness and date-range constraints when using Analytics.

## Limitations for Analytics

- The data in the dashboards updates every **24 hours**.
- Filters are limited to a maximum date range of **90 days**.

## Common questions

### How often does the data in the dashboards update?

The data in the dashboards updates every **24 hours**.

### What is the maximum date range allowed for filters?

Filters are limited to a maximum date range of **90 days**.

### Can I view data that is newer than the last 24-hour update?

The data in the dashboards updates every **24 hours**.

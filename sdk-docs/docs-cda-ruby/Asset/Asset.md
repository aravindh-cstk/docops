# Asset

## Asset

Asset class to fetch file details on Conentstack server.

## Properties

| Name | Type | Required | Default | Description |
|---|---|---|---|---|
| uid | String | No | — | Contentstack Asset UID for this asset |
| filename | String | No | — | Name of the asset. |
| url | String | No | — | URL to fetch/render the asset |
| content_type | String | No | — | Content Type for the asset. image/png, image/jpeg, application/pdf, video/mp4 etc. |
| filesize | String | No | — | Size of the asset. |
| tags | Array | No | — | Array of tags assigned to the asset. |

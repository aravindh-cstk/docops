# ResponseType

## ResponseType

The response type define the source from where the SDK retrieve the content.

## Properties

| Name | Type | Required | Default | Description |
|---|---|---|---|---|
| cache | ResponseType | No | — | This specifies response is from cache. |
| network | ResponseType | No | — | This specifies response is from network call. |

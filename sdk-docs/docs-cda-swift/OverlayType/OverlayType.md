# OverlayType

## OverlayType

The overlay parameter allows you to put one image on top of another. You need to specify the relative URL of the image as value for this parameter. SeeOverlay Settings

## Properties

| Name | Type | Required | Default | Description |
|---|---|---|---|---|
| align(OverlayAlign) | OverlayType | No | — | The overlay-align parameter lets you define the position of the overlay image. |
| repeat(OverlayRepeat) | OverlayType | No | — | The overlay-repeat parameter lets you define how the overlay image will be repeated on the given image. |
| size(Size) | OverlayType | No | — | The overlay-width parameter lets you define the width of the overlay image. Theoverlay-height parameter lets you define the height of the overlay image. |

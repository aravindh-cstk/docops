# APIError

## APIError

Information regarding an error received from Contentstack’s API.

## Properties

| Name | Type | Required | Default | Description |
|---|---|---|---|---|
| debugDescription | String | No | — | Error description. |
| errorMessage | String | No | — | Human readable error Message |
| errorCode | Int | No | — | Error Code from API. |
| errorInfo | ErrorInfo | No | — | More detailed error Information. |
| statusCode | Int | No | — | The HTTP status code. |

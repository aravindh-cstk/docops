# SystemFields

## SystemFields

System Fields are available fields for entities in Contentstack.

## Properties

| Name | Type | Required | Default | Description |
|---|---|---|---|---|
| uid | String | No | — | The unique identifier of the entity. |
| title | String | No | — | The title of the entity. |
| createdAt | Date | No | — | Describes the date of the entity is created. |
| updatedAt | Date | No | — | Describes the date of the entity is last updated. |

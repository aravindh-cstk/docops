# OverlayAlign

## OverlayAlign

The overlay-align parameter lets you define the position of the overlay image.

## Properties

| Name | Type | Required | Default | Description |
|---|---|---|---|---|
| top | OverlayAlign | No | — | Align the overlay image to the top of the actual image. |
| bottom | OverlayAlign | No | — | Align the overlay image to the bottom of the actual image. |
| left | OverlayAlign | No | — | Align the overlay image to the left of the actual image. |
| right | OverlayAlign | No | — | Align the overlay image to the right of the actual image. |
| middle | OverlayAlign | No | — | Align the overlay image to the middle (vertically) of the actual image. |
| center | OverlayAlign | No | — | Align the overlay image to the center (horizontally) of the actual image. |
| topLeft | OverlayAlign | No | — | Align the overlay image to the top-left of the actual image. |
| topRight | OverlayAlign | No | — | Align the overlay image to the top-right of the actual image. |
| bottomLeft | OverlayAlign | No | — | Align the overlay image to the bottom-left of the actual image. |
| bottomRight | OverlayAlign | No | — | Align the overlay image to the bottom-right of the actual image. |

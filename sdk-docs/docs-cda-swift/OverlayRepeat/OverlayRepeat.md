# OverlayRepeat

## OverlayRepeat

The overlay-repeat parameter lets you define how the overlay image will be repeated on the given image.

## Properties

| Name | Type | Required | Default | Description |
|---|---|---|---|---|
| both | OverlayRepeat | No | — | The overlay-repeat parameter for both. |
| horizontal | OverlayRepeat | No | — | The overlay-repeat parameter horizontal |
| verticle | OverlayRepeat | No | — | The overlay-repeat parameter verticle. |

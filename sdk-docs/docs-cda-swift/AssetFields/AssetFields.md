# AssetFields

## AssetFields

The Protocol for creating Asset model.

## Properties

| Name | Type | Required | Default | Description |
|---|---|---|---|---|
| createdBy | String? | No | — | Describes the unique identifier of user who created the entity. |
| updatedBy | String? | No | — | Describes the unique identifier of user who last updated the entity. |
| fileName | String | No | — | The name of the Asset. |
| fileSize | String | No | — | The size of the Asset. |
| fileType | String | No | — | The file type of the Asset. |
| url | String | No | — | The url for the Asset. |

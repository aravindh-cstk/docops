# Size

## Size

The Size parameter lets you dynamically resize the width and height of the output image by specifying pixels or percentage values.

## Properties

| Name | Type | Required | Default | Description |
|---|---|---|---|---|
| width | UInt | No | — | The width parameter lets you dynamically resize the width of the output image by specifying pixels or percentage values. |
| height | UInt | No | — | The height parameter lets you dynamically resize the height of the output image by specifying pixels or percentage values. |

# Orientation

## Orientation

The orient parameter lets you control the cardinal orientation of the given image.

## Properties

| Name | Type | Required | Default | Description |
|---|---|---|---|---|
| default | Orientation | No | — | Set image to default |
| flipHorizontal | Orientation | No | — | Flip image horizontally. |
| flipHorizontalVertical | Orientation | No | — | Flip image horizontally and vertically. |
| flipVerticle | Orientation | No | — | Flip image vertically. |
| flipHorizontalLeft | Orientation | No | — | Flip image horizontally and then rotate 90 degrees towards left. |
| right | Orientation | No | — | Rotate image 90 degrees towards right |
| flipHorizontalRight | Orientation | No | — | Flip image horizontally and then rotate 90 degrees towards right. |
| left | Orientation | No | — | Rotate image 90 degrees towards left. |

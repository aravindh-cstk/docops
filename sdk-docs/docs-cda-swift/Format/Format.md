# Format

## Format

The format parameter lets you converts a given image from one format to another.

## Properties

| Name | Type | Required | Default | Description |
|---|---|---|---|---|
| pjpg | Format | No | — | Progressive JPEG Format |
| jpeg | Format | No | — | JPEG format. |
| gif | Format | No | — | GIF format. |
| png | Format | No | — | PNG format. |
| webp | Format | No | — | WEBP format. |
| webply | Format | No | — | WEBP Lossy format. |
| webpll | Format | No | — | WEBP Lossless format. |

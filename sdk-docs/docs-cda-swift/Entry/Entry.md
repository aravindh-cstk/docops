# Entry

## Entry

An Entry is the actual piece of content created using one of the defined content types.

## Properties

| Name | Type | Required | Default | Description |
|---|---|---|---|---|
| cachePolicy | CachePolicy | No | — | Set cache policy for the entry request. |
| stack | Stack | No | — | Stack instance for Entry to be fetched |
| parameters | [String : Any] | No | — | URI Parameters. |
| queryParameter | [String : Any] | No | — | Query parameter |

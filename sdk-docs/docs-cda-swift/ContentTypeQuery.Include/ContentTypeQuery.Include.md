# ContentTypeQuery.Include

## ContentTypeQuery.Include

The `ContentTypeQuery.Include` is parameter for including count, Global Fields schema in result.

## Properties

| Name | Type | Required | Default | Description |
|---|---|---|---|---|
| count | ContentTypeQuery.Include | No | — | To include count in the response. |
| globalFields | ContentTypeQuery.Include | No | — | To include Global Fields schema in ContentType response, |
| all | ContentTypeQuery.Include | No | — | To include all ` ContentTypeQuery.Include` values. |

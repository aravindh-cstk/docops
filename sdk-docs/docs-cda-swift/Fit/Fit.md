# Fit

## Fit

The fit parameter enables you to fit the given image properly within the specified height and width. You need to provide values for the height, width and fit parameters

## Properties

| Name | Type | Required | Default | Description |
|---|---|---|---|---|
| bounds(Size) | Fit | No | — | If fit is set to bounds, it will constrain the given image into the specified height and width. |
| crop(Size) | Fit | No | — | If fit is set to crop, it will crop the given image to the defined height and width. |

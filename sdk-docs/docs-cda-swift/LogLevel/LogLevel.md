# LogLevel

## LogLevel

An enum describing the types of messages to be logged.

## Properties

| Name | Type | Required | Default | Description |
|---|---|---|---|---|
| none | LogLevel | No | — | Log nothing to the console. |
| error | LogLevel | No | — | Only log errors to the console. |
| info | LogLevel | No | — | Log messages when requests are sent, and when responses are received, as well as other useful information. |

# PublishType

## PublishType

This enable to sync entity with Published type.

## Properties

| Name | Type | Required | Default | Description |
|---|---|---|---|---|
| entryPublished | PublishType | No | — | To sync only Published Entries. |
| assetPublished | PublishType | No | — | To sync only Published Assets. |
| entryUnpublished | PublishType | No | — | To sync only Unpublished Entries. |
| assetUnpublished | PublishType | No | — | To sync only Unpublished Assets. |
| entryDeleted | PublishType | No | — | To sync only Deleted Entries. |
| assetDeleted | PublishType | No | — | To sync only Deleted Assets. |
| contentTypeDeleted | PublishType | No | — | To sync only only deleted content type Entries. |

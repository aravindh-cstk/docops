# LivePreviewConfig

## LivePreviewConfig

Configuration for enabled live preview to get live content data

## Properties

| Name | Type | Required | Default | Description |
|---|---|---|---|---|
| Enable | bool | No | false | To enable live preview |
| Host | string | No | — | Host for getting live preview content |
| PreviewToken | string | No | — | Preview Token for the stack to get content |

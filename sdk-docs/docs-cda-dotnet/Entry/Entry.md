# Entry

## Entry

An initializer is responsible for creating Entry object.

## Properties

| Name | Type | Required | Default | Description |
|---|---|---|---|---|
| Metadata | Dictionary<string, object> | No | — | Set key/value attributes of Metadata. |
| Object | Dictionary<string, object> | No | — | Set key/value attributes of an current entry instance. |
| Tags | object[] | No | — | Set array of Tags |
| Title | string | No | — | Title of an entry |
| Uid | string | No | — | This is Entry Uid of an entry. |

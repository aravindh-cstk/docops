# Asset

## Asset

Creates an instance of `Assets`. Use a `UID` to fetch a specific asset, or use the `AssetLibrary` class to retrieve all assets.

## Properties

| Name | Type | Required | Default | Description |
|---|---|---|---|---|
| Uid | string | Yes | — | Asset uid |
| Url | string | No | — | Asset URL |
| Description | string | No | — | Asset Description |
| FileName | string | No | — | File name for Asset |
| FileSize | string | No | — | File size of asset |
| Tags | object[] | No | — | Tags assign to asset |

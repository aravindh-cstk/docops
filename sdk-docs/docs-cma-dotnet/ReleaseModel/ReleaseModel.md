# ReleaseModel

## ReleaseModel

ReleaseModel for create or update release.

## Properties

| Name | Type | Required | Default | Description |
|---|---|---|---|---|
| Name | string | No | — | Release name to be set. |
| Description | string | No | — | Release description to be set. |
| Archived | bool | No | — | Set true to archive release |
| Locked | bool | No | — | Set true to lock the release content. |

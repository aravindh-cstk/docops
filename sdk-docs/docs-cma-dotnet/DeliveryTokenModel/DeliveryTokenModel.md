# DeliveryTokenModel

## DeliveryTokenModel

Delivery token scope for the token to be accessible.

## Properties

| Name | Type | Required | Default | Description |
|---|---|---|---|---|
| Name | string | No | — | Name for the delivery token to be created or updated. |
| Description | string | No | — | Description for the delivery token to be created or updated. |
| Scope | List<DeliveryTokenScope> | No | — | List of token scope for stack-level permissions you need to assign to the token. |

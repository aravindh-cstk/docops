# DeployModel

## DeployModel

Assets/Entries deploy model

## Properties

| Name | Type | Required | Default | Description |
|---|---|---|---|---|
| Action | string | No | — | Deploy action the the entry. |
| Environments | List<string> | No | — | Environment on which deploy assets/entries. |
| Locales | List<string> | No | — | List of locales for deployment. |
| ScheduledAt | string | No | — | Set date for future deployment. |

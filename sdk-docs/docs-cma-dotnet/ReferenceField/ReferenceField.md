# ReferenceField

## ReferenceField

Reference field class for adding reference field to content type.

## Properties

| Name | Type | Required | Default | Description |
|---|---|---|---|---|
| Plugins | List<string> | No | — | List of plugins to be added in reference field. |
| ReferenceTo | object | No | — | Set of content type to be added as reference in this field. |

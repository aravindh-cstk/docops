# UserInvitation

## UserInvitation

User invitation model to invite user to stacks or organisations.

## Properties

| Name | Type | Required | Default | Description |
|---|---|---|---|---|
| Email | string | No | — | User email for invitation to be sent. |
| Roles | List<string> | No | — | Roles from stack/organization to be assigned to user. |

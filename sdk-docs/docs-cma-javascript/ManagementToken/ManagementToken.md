# ManagementToken

## ManagementToken

[Management Tokens](/docs/developers/create-tokens/about-management-tokens) are tokens that provide you with read-write access to your stack's content. When used in conjunction with the stack API key, they authorize Content Management API (CMA) requests for the purpose of managing your stack's content.

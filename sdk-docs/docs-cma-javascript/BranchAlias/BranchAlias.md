# BranchAlias

## BranchAlias

Branches efficiently present independent workspaces where developers and content managers can work parallely on content models and content. It helps sync the development activities of websites.

A branch [alias](/docs/developers/branches/about-aliases/) is a custom name given to a specific branch in a stack to make referencing easier, especially when working with multiple branches.

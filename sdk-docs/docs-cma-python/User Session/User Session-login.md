## login

The login method allows you to sign in to your Contentstack account.

| Name | Type | Required | Default | Description |
|---|---|---|---|---|
| email | str | Yes | — | The email ID of the user |
| password | str | Yes | — | The password of the user |
| tfa_token | str | No | — | Token generated by user’s authenticator app or received via SMS |

Returns:
Type
Response Object

```
import contentstack_management
client = contentstack_management.Client()
client.login(username, password, tfa_token)
```

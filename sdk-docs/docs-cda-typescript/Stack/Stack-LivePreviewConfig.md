## LivePreviewConfig

Configuration settings to enable live preview functionality and fetch real-time content data.

| Name | Type | Required | Default | Description |
|---|---|---|---|---|
| enable | boolean | No | false | Specifies whether to enable the live preview feature. |
| host | string | No | — | Specifies the host domain used to retrieve live preview content. |
| preview_token | string | No | — | Token required to fetch live preview content from the stack. |

# Assets

## Assets

Creates an instance of `Assets`.

Retrieves all assets of a stack by default. To retrieve a single asset, specify its UID.

## Properties

| Name | Type | Required | Default | Description |
|---|---|---|---|---|
| uid | string | No | — | UID of asset you want to retrieve |

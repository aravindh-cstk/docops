# Config

## Config

Stack configuration for setting stack details.

## Properties

| Name | Type | Required | Default | Description |
|---|---|---|---|---|
| host | NSString | No | cdn.contentstack.io | Host name of Contentstack api server. |
| region | ContentstackRegion | No | — | You can choose from seven regions namely NA, EU, AU, Azure NA, Azure EU, GCP NA and GCP EU. |
| version | NSString | No | v3 | API version of Contentstack api server. |
| branch | NSString | No | — | Branch id for getting content from Stack |

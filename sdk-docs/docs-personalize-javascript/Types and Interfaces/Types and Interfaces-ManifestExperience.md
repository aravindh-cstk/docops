## ManifestExperience 

The `ManifestExperience ` type alias defines the structure of an experience within a personalization manifest.

| Name | Type | Required | Default | Description |
|---|---|---|---|---|
| shortUid | string | No | — | The unique identifier for the experience. |
| activeVariantShortUid | string | No | — | The unique identifier for the active variant associated with the experience; it can be null if no variant is active. |

# [Second level navigation] - Kickstarts

This page introduces Contentstack Kickstarts and provides links to Kickstart guides. It is intended for developers who want to start a new website, app, or integration using pre-built templates and implementation resources.

### Listing page

Contentstack Kickstarts

#### Introduction

Kickstarts are pre-built, customizable templates designed to help you launch faster with Contentstack. Whether you are setting up a new website, app, or integration, Kickstarts provide ready-to-use structures, best practices, and sample content to accelerate development. With Kickstarts, you can go from idea to execution with less setup time, so you can focus on building standout digital experiences.

Contentstack Kickstart guides are a collection of implementation resources that include both step-by-step guides and starter code needed to connect to Contentstack. This Kickstart walks you through SDK initialization, along with the setup for Live Preview and Visual Builder, helping you lay a strong foundation for your project.

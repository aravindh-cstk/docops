## Common questions

1. **What is this page used for?**  
   It provides a categorized list of Content Migration documentation links, including developer guides, setup guides, and migration walkthroughs.

2. **Which guide should I start with to install the framework?**  
   Use the relevant “Set Up Content Migration Framework” guide for your environment (macOS, Windows, or Docker).

3. **Where do I find connector documentation for my legacy CMS?**  
   Look under “Developer Guides” for the CMS Connector API links (for example, Sitecore, Contentful, or Wordpress).

4. **How do I find step-by-step migration instructions?**  
   Use the “Perform Content Migration” section to select the guide for your source CMS (for example, Sitecore to Contentstack or AEM to Contentstack).

# [JavaScript Delivery] - Build Apps Using Angular SDK

This page explains how to build Angular applications using Contentstack’s Angular SDK (Angular 2 and above) to fetch and power app content from Contentstack. It is intended for developers building Angular apps who need to integrate Contentstack content delivery.

## Build Apps Using Angular SDK

Contentstack provides [JavaScript SDK](https://github.com/contentstack/contentstack-javascript) using which you can create applications based on JavaScript, Angular, and React. This SDK lets you power your app’s content by Contentstack.

However, if you wish to create apps based on Angular, you can download and use the [Angular SDK](https://github.com/contentstack/angular-contentstack-service) (for Angular 2 and above) and start building your apps. With this SDK, you can build Angular apps and fetch content for the app from Contentstack.

## Common questions

**How is the Angular SDK different from the JavaScript SDK?**  
The JavaScript SDK supports applications based on JavaScript, Angular, and React, while the Angular SDK is specifically intended for Angular (Angular 2 and above).

**Which Angular versions are supported by the Angular SDK?**  
The Angular SDK is for Angular 2 and above.

**Where can I find the SDK repositories?**  
JavaScript SDK: https://github.com/contentstack/contentstack-javascript  
Angular SDK: https://github.com/contentstack/angular-contentstack-service

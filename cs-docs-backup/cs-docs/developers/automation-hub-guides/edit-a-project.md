# [Automations guides and connectors] - Edit a Project

This page explains how to edit a Project’s primary details (Name and Description) from the automation settings page. It is intended for users managing projects in Automation Hub and should be used when you need to update an existing project’s basic information.

## Edit a Project

You can edit the primary details of a Project, i.e., its Name and Description, from the automation settings page.

To edit a project, perform the steps given below:
- In the left navigation panel, click **Settings**.
- Enter the **Project Name** and **Description** that you want to edit and then click the **Save Changes** button.

**Note: **An organization's owner and admin can edit and view all projects within automate, while organization members can only view and edit their projects.

## Common questions

### Who can edit and view projects?
An organization's owner and admin can edit and view all projects within automate, while organization members can only view and edit their projects.

### What project details can be edited from the automation settings page?
You can edit the primary details of a Project, i.e., its Name and Description.

### Where do I go to edit a project?
In the left navigation panel, click **Settings**.

### How do I save my edits to a project?
Enter the **Project Name** and **Description** that you want to edit and then click the **Save Changes** button.

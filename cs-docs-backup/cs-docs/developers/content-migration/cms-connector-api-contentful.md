# [Content Migration] - CMS Connector API - Contentful

This page describes the CMS Connector API for Contentful and explains what the `migration-contentful` package is for, who should use it (developers migrating from Contentful to Contentstack), and when to reference it (while planning or executing a Contentful-to-Contentstack migration).

## CMS Connector API - Contentful

This document outlines the functionalities and usage of the `migration-contentful` package, which is designed to simplify the process of migrating data from Contentful to Contentstack CMS. It provides key utility functions for extracting content types, mapping fields, and handling locales, ensuring a smooth and efficient migration process.

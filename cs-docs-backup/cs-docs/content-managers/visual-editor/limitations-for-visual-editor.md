# [Visual Editor] - Limitations for Visual Editor

This page lists the limitations and compatibility requirements for the Visual Editor feature. It is intended for content managers and developers who need to understand prerequisites and constraints before enabling or using Visual Editor in their organization.

## Limitations for Visual Editor

- The Visual Editor feature is only available if your organization has the Live Preview and Visual Editor plan enabled. Please contact the [support](mailto:support@contentstack.com) team to activate this plan for your organization.
- Visual Editor does not support websites generated using third-party builders or CMS platforms to produce HTML.
- The feature is not compatible with mobile applications.
- You cannot preview or edit entry content without the necessary permissions for accessing entries, specific environments, or published languages.
- Your website should support `Iframes` to prevent CORS errors during preview.

A: Visual Editor does not support websites generated using third-party builders or CMS platforms to produce HTML.

**Q: Can I use Visual Editor in a mobile app?**  
A: The feature is not compatible with mobile applications.

A: You cannot preview or edit entry content without the necessary permissions for accessing entries, specific environments, or published languages.

# Log out of your account

**DELETE** `/user-session`

The Log out of your account call is used to sign out the user of Contentstack account.

## Headers

- **authtoken** (required)
  Default: `your_authtoken`

## Sample Response

```json
{
	"notice": "You've logged out successfully!"
}
```

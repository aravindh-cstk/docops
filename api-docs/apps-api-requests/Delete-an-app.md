# Delete an app

**DELETE** `/apps/{app_uid}`

## URL Parameters

- **app_uid** (required)
  The UID of the app to be updated.

## Headers

- **authtoken** (required)
  Enter your management token.
  Default: `your_authtoken`
- **organization_uid** (required)
  The UID of the organization.
  Default: `your_organization_uid`
